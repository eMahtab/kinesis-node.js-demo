# kinesis-node.js-demo

# Instructions


```
git clone https://github.com/eMahtab/kinesis-node.js-demo.git
cd kinesis-node.js-demo
npm install


Create stream
AWS_PROFILE={YOUR_AWS_PROFILE_NAME} node create.js

Write Records to Stream
AWS_PROFILE={YOUR_AWS_PROFILE_NAME} node write.js

Read Records to Stream
AWS_PROFILE={YOUR_AWS_PROFILE_NAME} node read.js

Delete Stream
AWS_PROFILE={YOUR_AWS_PROFILE_NAME} node delete.js

```
